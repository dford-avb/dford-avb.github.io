Openmage Info API

Allows you to get information about the current Openmage installation.

Module: Mage_Core

Resource: core_openmage

Aliases: openmage

Method:
  • core_openmage.info (SOAP V1)
  • openmageInfo (SOAP V2)

Allows you to retrieve information about Openmage version and edition.

Aliases: openmage.info

Arguments:

Type
Name
Description
string
sessionId
Session ID

Returns:

Type
Name
Description
string openmage_version Openmage version
string openmage_edition Openmage edition (Community, Professional, Enterprise)

Examples

Request Example SOAP V1
$proxy = new SoapClient('http://openmagehost/api/soap/?wsdl');
$sessionId = $proxy->login('apiUser', 'apiKey');
$openmageInfo = $proxy->call(
    $sessionId,
    'openmage.info'
);
Request Example SOAP V2
$proxy = new SoapClient('http://openmagehost/api/v2_soap/?wsdl'); // TODO : change url
$sessionId = $proxy->login('apiUser', 'apiKey'); // TODO : change login and pwd if necessary

$result = $proxy->openmageInfo($sessionId);
var_dump($result);
Request Example SOAP V2 (WS-I Compliance Mode)
$proxy = new SoapClient('http://openmagehost/api/v2_soap/?wsdl'); 

$sessionId = $proxy->login((object)array('username' => 'apiUser', 'apiKey' => 'apiKey')); 
 
$result = $proxy->openmageInfo((object)array('sessionId' => $sessionId->result));   

var_dump($result->result);
