The following APIs allow you to access additional Openmage information.

Module: Store View API

Allows you to retrieve information on the store view.

Resource Name: Store

Methods:
  • store.info - Get information about a store view
  • store.list - Get the list of store views

Module: Mage_Core API

Allows you to get information about the current Openmage installation.

Resource Name: core_openmage

Aliases: openmage

Method:
  • core_openmage.info
Create the Openmage file system owner
