Overview

This page discusses how to get the sample data for:

  • Openmage Community Edition (CE) 1.6, 1.7, or 1.8
  • Openmage Community Edition (CE) 1.9.0.0
  • Openmage Community Edition (CE) 1.9.1.0–1.9.2.0
Important: Openmage CE 1.9 and later use different sample data from the other versions in the preceding list. Make sure you use the correct version of sample data with your version of Openmage CE.

Install Sample Data Before You Install Openmage

Openmage CE sample data must be installed before you install Openmage CE.

Getting the Sample Data

The sample data is provided in different formats for your convenience. Archives for each version have exactly the same content (they differ only by compression method).

  • Openmage CE 1.9.1.0–1.9.2.0

Installing the Sample Data

Detailed instructions for installing sample data for Openmage CE 1.6–1.9 can be found on the Openmage Knowledge Base.
