Overview

We're happy you chose to install or upgrade your Openmage software. We're going to help you to install or to upgrade to the following versions:

  • Openmage Community Edition (CE) 1.8 or 1.9
  • Openmage Enterprise Edition (EE) 1.13. or 1.14

These releases include a number of improvements:

  • Openmage Community Edition (CE) Release Notes
  • Openmage Enterprise Edition (EE) Release Notes
  • Openmage CE and EE documentation home page, which also has information about new features
Important: Use CE 1.9.1 or EE 1.14.1 or later for all new EE installations and upgrades to get the latest fixes, features, and security updates.

The sections that follow get you started on your installation or upgrade. Detailed, step-by-step procedures are discussed in the following articles:

  • Installing Openmage CE or Openmage EE
  • Upgrading to Openmage CE or EE

Getting Openmage CE or EE

The following table discusses where to get the upgrade image and optional sample data.

Openmage edition Upgrade image location
Openmage CE www.openmagecommerce.com/download
Openmage EE Use the following steps:
  1. Go to www.openmage.com
  2. In the top horizontal navigation bar, click My Account.
  3. Log in with your Openmage username and password.
  4. In the left navigation bar, click Downloads.
  5. In the right pane, click Openmage Enterprise Edition > Release for the software or Sample Data for the optional sample data.
  6. Follow the instructions on your screen to complete the Openmage EE download.
  7. Transfer the installation archive to your development system.
